# 三、Linux目录和文件管理



## Linux文件属性

“**文件属性**”描述了文件的**类型**、**硬链接数**、 **权限**、**User**、**Group**、**大小(Byte)**、**最后修改时间**、**文件名**等信息。

适用 `ls -l`	查看文件属性

如 

```sh
rye@Hhhjt:~/LearningShell$ ls -l
total 4
-rw-r--r-- 1 rye rye 38 May  5 12:26 hello.sh
```

| **文件类型**     |     **`-`**     |
| :--------------- | :-------------: |
| **权限**         | **`rw-r--r--`** |
| **硬链接数**     |      **1**      |
| **User**         |     **rye**     |
| **Group**        |     **rye**     |
| **大小**         |   **38 Byte**   |
| **最后修改时间** | **May 5 12:26** |
| **文件名**       |  **hello.sh**   |



### Linux常见文件类型

1. **普通文件**

| `-`  | 普通文件 |
| :--: | :------: |

2. **目录文件**  (***Directory***)

	|  d   | 目录文件 |
	| :--: | :------: |

3. **设备文件**

	| **b** |      **块设备(如磁盘)**       |
	| :---: | :---------------------------: |
	| **c** | **字符设备 (如打印机和终端)** |

4. **管道文件**

	|  p   | 管道文件 |
	| :--: | :------: |

5. **链接文件**

	|         l          |   软链接   |
	| :----------------: | :--------: |
	| **和链接文件一致** | **硬链接** |

	 

### Linux文件权限(*简略*)

**`rw-r--r--`**分为三组，分别描述**User Group Others**

| **r** |  **可读**  |
| :---: | :--------: |
| **w** |  **可写**  |
| **x** | **可执行** |
| **-** | **无权限** |
